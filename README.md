#  常用刷课软件
本项目旨在帮助辽石化的同学们轻松获得课程平时分。可刷内容包括：词达人，we learn，知到（智慧树），学习通，以及学xx国。
点击下载需要刷课的zip文件后需要解压，文件夹内均有使用方法。
词达人和welearn为本地软件免费刷，知到(智慧树)和学习通为平台收费刷课(每课1r以下)，学xx国为电脑网页油猴脚本免费刷(具体使用方法在文件夹，每天可刷45积分)。
